# 配置说明

本文按 `.env.example` 分组说明常用配置。

## Base

| 配置 | 默认值 | 说明 | 生产建议 |
| --- | --- | --- | --- |
| `APP_NAME` | `BuildingAI` | 应用名称 | 按品牌修改 |
| `APP_DOMAIN` | 空 | 应用域名 | 生产必须设置为真实 HTTPS 域名 |

## Server

| 配置 | 默认值 | 说明 | 生产建议 |
| --- | --- | --- | --- |
| `SERVER_PORT` | `4090` | API 和静态资源服务端口 | 按部署端口配置 |
| `SERVER_CORS_ENABLED` | `true` | 是否启用 CORS | 按需启用 |
| `SERVER_CORS_ORIGIN` | `*` | 允许的跨域来源 | 生产不要使用 `*` |
| `SERVER_SHOW_DETAILED_ERRORS` | `false` | 是否显示详细错误 | 生产保持 `false` |
| `SERVER_IS_DEMO_ENV` | `false` | 是否演示环境 | 演示站可启用 |

## PM2

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| `PM2_APP_NAME` | `buildingai` | PM2 应用名 |
| `PM2_INSTANCES` | `1` | 实例数量 |
| `PM2_EXEC_MODE` | `cluster` | 执行模式 |
| `PM2_MAX_MEMORY` | `1G` | 内存重启阈值 |
| `PM2_WATCH` | `false` | 是否监听文件变化 |
| `PM2_LOG_DIR` | `../../logs/pm2` | PM2 日志目录 |
| `PM2_AUTORESTART` | `true` | 是否自动重启 |

## JWT

| 配置 | 默认值 | 说明 | 是否敏感 |
| --- | --- | --- | --- |
| `JWT_SECRET` | `buildingai` | JWT 签名密钥 | 是 |
| `JWT_EXPIRES_IN` | `30d` | token 有效期 | 否 |

生产环境必须替换 `JWT_SECRET`。

## Database

| 配置 | 默认值 | 说明 | 是否敏感 |
| --- | --- | --- | --- |
| `DB_TYPE` | `postgres` | 数据库类型 | 否 |
| `DB_HOST` | `localhost` | 数据库主机 | 否 |
| `DB_PORT` | `5432` | 数据库端口 | 否 |
| `DB_USERNAME` | `postgres` | 数据库用户名 | 是 |
| `DB_PASSWORD` | `postgres` | 数据库密码 | 是 |
| `DB_DATABASE` | `buildingai` | 数据库名 | 否 |
| `DB_SYNCHRONIZE` | `false` | 生产 schema 同步 | 否 |
| `DB_DEV_SYNCHRONIZE` | `true` | 开发 schema 同步 | 否 |
| `DB_LOGGING` | `true` | 是否打印数据库日志 | 否 |

生产环境保持 `DB_SYNCHRONIZE=false`，并在升级前备份数据库。

## Redis

| 配置 | 默认值 | 说明 | 是否敏感 |
| --- | --- | --- | --- |
| `REDIS_HOST` | `localhost` | Redis 主机 | 否 |
| `REDIS_PORT` | `6379` | Redis 端口 | 否 |
| `REDIS_USERNAME` | 空 | Redis 用户名 | 是 |
| `REDIS_PASSWORD` | 空 | Redis 密码 | 是 |
| `REDIS_DB` | `0` | Redis DB | 否 |
| `REDIS_TTL` | `60*60*24` | 默认过期时间 | 否 |
| `REDIS_MAX_RECONNECT_ATTEMPTS` | `5` | 最大重连次数 | 否 |
| `REDIS_RECONNECT_DELAY` | `3000` | 重连间隔毫秒 | 否 |

生产环境必须设置 Redis 密码，不要公网裸露 Redis。

## Cache

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| `CACHE_TTL` | `86400` | 内存缓存默认 TTL |
| `CACHE_MAX_ITEMS` | `1000` | 内存缓存最大项目数 |

## Logs

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| `LOG_LEVELS` | `error,warn,debug,fatal` | 控制台日志级别 |
| `LOG_WRITE_LEVELS` | `error,warn,debug,fatal,log` | 文件写入日志级别 |
| `LOG_TO_FILE` | `true` | 是否写入文件 |
| `LOG_DATABASE_SCHEMA` | `false` | 是否打印数据库 schema |
| `LOG_CLEAN_INTERVAL` | `3` | 日志清理间隔 |

## Web

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| `VITE_DEVELOP_APP_BASE_URL` | `http://localhost:4090` | 开发 API 地址 |
| `VITE_PRODUCTION_APP_BASE_URL` | 空 | 生产 API 地址 |
| `VITE_APP_WEB_API_PREFIX` | `/api` | 前台 API 前缀 |
| `VITE_APP_CONSOLE_API_PREFIX` | `/consoleapi` | 后台 API 前缀 |

## Docker

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| `NPM_REGISTRY_URL` | `https://registry.npmmirror.com` | 容器内 npm registry |
| `DOCKER_CONTAINER_SUFFIX` | 空 | 容器名后缀 |
| `DOCKER_MEMORY_LIMIT` | `6144M` | 容器内存限制 |
| `DOCKER_CPU_LIMIT` | `2.0` | 容器 CPU 限制 |
| `DOCKER_MEMORY_RESERVATION` | `512M` | 容器内存预留 |
| `POSTGRES_EXTERNAL_PORT` | 空 | PostgreSQL 宿主机端口 |
| `REDIS_EXTERNAL_PORT` | 空 | Redis 宿主机端口 |

## 配置同步

运行：

```bash
pnpm sync-env
```

同步脚本会：

- 读取 `.env.example`。
- 保留 `.env` 中已有值。
- 追加新增配置项。
- 移除废弃配置项。
- 忽略少量环境专属配置。

建议每次拉取新版本后运行一次。
