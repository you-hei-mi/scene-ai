## 版本备注

1. 当前版本号来自根目录 `package.json`：`26.1.1`。
2. 本轮部署工作新增了总控部署实施指南，并完成虚拟机 Docker Compose 部署验证。
3. 虚拟机当前通过 `192.168.232.100:4090`
   提供应用服务；PostgreSQL 与 Redis 仅建议在受控内网范围内开放。
4. `/consoleapi/health/db` 需要登录态会返回
   `401 Unauthorized`，本轮以容器健康状态、应用日志和基础健康检查确认 PostgreSQL 与 Redis 均已连接。
5. 本地开发调试请访问 `http://localhost:4091`，不要访问虚拟机生产地址；本地前端通过
   `VITE_DEVELOP_APP_BASE_URL=http://localhost:4090` 请求本地 API。
6. 扩展前端开发需要同时启动扩展自己的 Vite 服务；当前 `simple-blog` 使用
   `http://localhost:5173/extension/simple-blog`，并通过
   `VITE_EXTENSION_DEV_BASE_URLS=simple-blog=http://localhost:5173` 接入宿主 iframe。
