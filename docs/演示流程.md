# 演示流程

本文用于产品演示、售前演示和功能验收。

## 1. 初始化安装

访问：

```text
http://localhost:4090/install
```

演示要点：

- 欢迎页。
- 创建管理员账号。
- 配置站点名称、描述、Logo、图标和主题。
- 初始化成功后进入系统。

关键路径：

- `/install`
- `apps/web-nuxt/app/pages/install.vue`

## 2. 登录与控制台概览

访问：

```text
http://localhost:4090/login
```

登录后进入：

```text
/admin
```

演示要点：

- 后台管理统一挂载在 `/admin/*`。
- 管理菜单和权限来自当前登录用户。
- 管理员可进入数据分析、公告管理、模型管理、系统设置和用户管理等模块。

## 3. 配置模型与服务商

访问：

```text
/admin/models
```

演示要点：

- 新增或启用 AI 服务商。
- 配置模型参数。
- 进入 `/admin/settings` 管理密钥和系统配置。
- 进入 `/mcp` 管理 MCP 服务和工具。

## 4. 创建和配置 Agent

访问：

```text
/agents
```

演示步骤：

1. 创建 Agent。
2. 选择直接创建、Coze 或 Dify。
3. 进入 `/agents/[id]` 查看和编辑 Agent 详情。
4. 配置角色提示词。
5. 配置表单变量。
6. 绑定知识库。
7. 配置上下文、引用、MCP 工具、联网搜索和文件上传。
8. 配置开场白、推荐问题、自动追问、快捷指令和聊天头像。
9. 配置模型路由和语音能力。
10. 使用调试预览验证效果。

## 5. Agent 聊天体验

访问：

```text
/agents/[id]
```

或通过通用聊天入口：

```text
/chat
```

演示要点：

- 发送问题并展示流式回复。
- 演示历史会话。
- 演示重新生成、编辑消息、点赞/点踩。
- 演示语音输入、语音朗读和文件上传。
- 如果已绑定知识库，展示引用和知识库增强回答。

## 6. Agent 发布

访问 Agent 详情页后，可通过发布设置对外分享。

演示要点：

- 开启网页访问。
- 复制公开链接。
- 展示 iframe 嵌入代码。
- 展示浮窗脚本。
- 如需广场发布，演示发布到广场和后台审核流程。

## 7. 知识库创建与问答

访问：

```text
/datasets
```

演示步骤：

1. 创建或打开知识库 `/datasets/[id]`。
2. 上传文档或从 URL 导入。
3. 查看文档列表、分类 tab、预览与批量操作。
4. 等待解析和向量化完成。
5. 在问答面板提问。
6. 展示知识库回答、引用片段和检索效果。
7. 演示成员管理、标签编辑、文档转移和发布到知识广场。

## 8. 控制台审核与运营

常用入口：

- 数据分析与仪表板: `/admin`、`/admin/analytics`
- 公告管理: `/admin/announcements`
- 模型服务商: `/admin/models`
- 系统设置: `/admin/settings`
- 用户管理: `/admin/users`
- MCP 管理: `/mcp`
- Agent 广场: `/agents`
- 知识库: `/datasets`
- 聊天记录: `/chat`
- 应用广场: `/apps`
- 通知中心: `/notifications`

## 演示前检查

- 服务健康检查通过: `/consoleapi/health`。
- 已完成初始化安装。
- 已配置可用 Provider、模型和密钥。
- 如果演示知识库，至少准备一个已完成向量化的文档。
- 如果演示发布链接，确认 `APP_DOMAIN` 或本地访问地址正确。
